Detailed Description

The ConfigurationDatabase provides access to the contents of the configuration files in the config/ directory.

The load method should be called once to initialize the database, passing the location of the config/ directory. All configuration files with extension ".cfg" are passed to a lua interpreter instance for parsing / execution.
