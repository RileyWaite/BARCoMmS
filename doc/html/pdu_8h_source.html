6 /* FILE: pdu.h -- specs for a module that handles PDUs (e.g. assembles,
7  * disassembles, and displays PDUs).
8  * ORIGINAL PROGRAMMER: Tim Ray 301-286-0581
9  * BIG PICTURE: The intent is to allow most of the CFDP library to work
10  * with PDUs as if they were standard "C" data structures. Each incoming
11  * (raw) PDU is immediately converted to a C data structure, and each
12  * outgoing PDU is converted from the internal data structure to raw bytes
13  * at the last possible moment.
14  * Hopefully, this allows the library source code to be easier to read,
15  * and, perhaps, more efficient as well.
16  * SUMMARY: There are routines for converting from C data structures to
17  * raw PDU bytes (assembly), and vice versa (disassembly).
18  * Plus, there are a few special purpose routines.
19  * CHANGES:
20  * 2007_07_31 Tim Ray
21  * - Official release. (see pre_release_changes.log for earlier changes)
22  */
